# 📡 Landing Page - Telecom Rápida

Landing page institucional desenvolvida para uma provedora de internet fictícia. Projeto criado durante meus estudos de HTML, CSS e JavaScript.

## 🔗 Demonstração

[Clique aqui para ver o site ao vivo](https://KevLps.github.io/landing-page-telecom/)

## 🛠️ Tecnologias utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Flexbox, Grid, responsividade, animações
- **JavaScript** - Modal personalizado, localStorage, rolagem suave

## ✨ Funcionalidades

- ✅ Design responsivo (funciona em celular, tablet e desktop)
- ✅ Cards de planos com hover effect
- ✅ Card destaque "Mais popular"
- ✅ Modal bonito (substitui o alert feio do navegador)
- ✅ Formulário de contato com validação
- ✅ Salvamento automático dos dados no localStorage
- ✅ Rolagem suave ao clicar nos links do menu

## 🧠 O que aprendi com este projeto

- Estruturar uma landing page completa
- Aplicar CSS Grid e Flexbox para alinhamentos
- Criar um modal personalizado com JavaScript
- Salvar e recuperar dados no localStorage
- Fazer deploy no GitHub Pages

O projeto foi desenvolvido durante meus estudos de HTML e CSS. Utilizei JavaScript para adicionar interatividade (modal, localStorage e formulário) com auxílio de documentação e tutoriais, estou em constante evolução.
